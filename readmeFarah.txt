LOGIN PAGE (login.html):

What it does:

Provides an admin login interface for SupplyLink.

Allows admins to enter their username and password.

Uses the shared top banner for navigation (Home, Signup, Login).

How to use:
Open login.html.
Enter Admin Username in the input field.
Enter Password in the password field.

Click the Login button to submit.

Authentication handled through /auth.js.

Features:

Clean centered login card.
Uses .login-banner for title header inside the form.
Responsive width (40%, adjusts on smaller screens).
Styled inputs and button with hover effects.
Integrated navigation banner for consistent layout.

SIGNUP PAGE (signup.html):

What it does:
Lets new users register an account for SupplyLink.
Collects basic information such as username and password.
Styled and structured similarly to the login page.

How to use:

Open signup.html.

Fill in all required form fields.

Click the Signup button to submit.

Successful signup redirects to login.

Features:

Styled form using .signup-form with white card layout.

Real-time validation indicators (:valid, :invalid).

Input focus effects (border-color accent).

Responsive design for all screen sizes.
